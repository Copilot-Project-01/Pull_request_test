# Pull_request_test

## Simple Copilot MCP Server

This repository includes a Python implementation of a simple Model Context Protocol (MCP) server for GitHub Copilot.

### Quick Start

```bash
# Run the MCP server
python3 mcp_server.py

# Run tests
python3 test_mcp_server.py
```

For detailed documentation, see [MCP_SERVER_README.md](MCP_SERVER_README.md)
